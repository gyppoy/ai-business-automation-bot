# 🤖 AI Business Automation Bot

An intelligent AI-powered chatbot that:
- Captures leads automatically
- Understands user intent
- Extracts structured data (name, budget, requirements)
- Supports multi-user conversations
- Stores leads for business use

## 🚀 Features
- FastAPI backend
- Local AI (Ollama - LLaMA3)
- Multi-user memory system
- AI-based lead extraction
- JSON lead storage

## 🛠 Tech Stack
- Python
- FastAPI
- Ollama (LLaMA3)
- Requests

pip install -r requirements.txt
